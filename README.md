# AiLang Linux Target

Official AiLang target package for Linux.

This target is for normal Linux desktop/server environments. AiOS shell-less
images live in `ailang-target-aios`.

## Package

```text
packages/target-linux
```

## Status

Initial target repository scaffold. Package discovery is enabled; platform
tooling currently returns deterministic not-yet-implemented diagnostics until
the Linux implementation is moved in.
